# Changelog Terbaru

Berikut adalah daftar 5 perubahan terakhir pada sistem Akrab Auto:

1. **Unifikasi & Perapian UI Notifikasi**  
   Menyeragamkan seluruh pesan notifikasi admin (Otomatis, Manual, Webhook, Ghost Stock) dengan layout simetris, perataan teks lurus, dan garis pemisah lebar agar lebih profesional.

2. **Pembaruan Format Tanggal dan Waktu**  
   Mengubah format waktu ISO yang kaku menjadi format yang lebih manusiawi (DD-MM-YYYY HH:mm:ss) pada seluruh interaksi bot dan notifikasi.

3. **Penyempurnaan UI Cek Stok**  
   Menyejajarkan tanda titik dua (:) pada daftar stok dan memindahkan emoji peringatan ke dalam kurung (⚠️ Ghost: X) agar tampilan lebih simetris dan rapi.

4. **Perbaikan Logika Tombol Retry**  
   Mengembalikan fungsi tombol Retry (♻️) agar hanya muncul pada pesanan dengan status ERROR untuk menjaga konsistensi alur kerja.

5. **Pembaruan UI Menu List (Satu Baris & Lebar)**  
   Memperpanjang garis pemisah antar pesanan dan menyatukan semua tombol aksi ke dalam satu baris horizontal agar tampilan lebih ramping dan luas.
